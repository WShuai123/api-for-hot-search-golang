## 运行

`go run main.go`

默认端口为1111，可以自己改。

浏览器打开`ip:1111`

具体路径看`main.go`中的路由表

比如访问百度的热搜就是：`ip:1111/baidu`

查看全部app的热搜为`ip:1111/all`

建议`go build`为可执行文件。
